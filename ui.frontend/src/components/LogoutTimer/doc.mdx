# LogoutTimer

import { Story } from '@storybook/addon-docs';

Componente que manipula o tempo de "refresh" da página Home

## Propriedades

### Text

Texto renderizado no Botão

padrão: `Continuar Navegando`

### fontSize

tamanho da fonte exibida no botão

padrão: `12`

### seconds

valor em segundos até que o usuário seja desconectado da página Home
