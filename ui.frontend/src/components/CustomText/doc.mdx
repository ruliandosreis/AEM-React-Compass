# CustomText

import { Story } from '@storybook/addon-docs';

Componente de texto flexível

## Propriedades


### text

Texto renderizado na tela

padrão: `Texto padrão de exemplo`

### fontSize

Tamanho da fonte do texto

padrão: `48`

### color

Cor do texto renderizado

padrão: `#000000`

### margin

Propriedade Margin do css do texto

padrão: `0`

### family

Propriedade font-family do css do texto

padrão: `Mark-Pro`

### weight

Propriedade weight do css do texto

padrão: `700`
